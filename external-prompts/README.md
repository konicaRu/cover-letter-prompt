# external-prompts

Промпты и методология для сторонних нейросетей (ChatGPT, Claude и т.п.), которые
должны генерировать сопроводительные письма вне этого проекта.

## Что внутри
- [`cover-letter-chatgpt.md`](cover-letter-chatgpt.md) — готовый системный промпт
  (образец). Вставляется в Instructions проекта ChatGPT; факты берутся из
  загруженных в проект файлов резюме и Jira. В чат подаётся текст вакансии →
  на выходе 3 варианта письма.
- [`building-rules.md`](building-rules.md) — правила, по которым собран этот промпт:
  выжимка из 14 правок (C-001…C-014) и принципов проекта. Используется, чтобы
  пересобрать/доработать промпт под другого кандидата или сервис.
- [`gpt5-prompting-guide.md`](gpt5-prompting-guide.md) — конспект официального гайда
  OpenAI по промптингу GPT-5 (структура промпта, контроль «усердия», self-check
  и т.д.), по которому оформлен сам промпт.

## Источник методологии
Основной движок и история правок — в корне проекта (`prompt/`, `CLAUDE.md`,
`prompt/corrections.md`). Эта папка — «экспортный» дистиллят для внешних сервисов.
