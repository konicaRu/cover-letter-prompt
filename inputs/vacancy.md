# Вакансия: Аналитик данных — «Восход» (фарм-дистрибуция)

«Восход» — компания на рынке 20+ лет. Нужен аналитик данных с опытом построения аналитики
с нуля: создать единую систему сбора, обработки, анализа и визуализации данных из множества
источников. Основная задача — собрать данные из 1С, МДЛП, АльФарм, IQVIA, CRM, Brand Analytics,
Metrics, отчётов АС и медпредставителей, привести к единой структуре и автоматизировать.

## Ключевые задачи
- Сбор/обработка/анализ из разнородных источников: Postgres, ClickHouse, Excel/CSV, 1С, внешние.
- Данные по продажам, остаткам, движению лекарств, дистрибуции, аптекам, складам, финпоказателям.
- Работа с МДЛП (мониторинг движения лекарственных препаратов).
- Регулярная управленческая отчётность.
- BI-дашборды (Power BI, Tableau, Superset или другие — на выбор).
- Автоматизация ETL, единое хранилище данных.

## Требования
- Аналитик данных / BI-аналитик от 3 лет.
- Уверенный Python (pandas).
- БД и SQL (сложные запросы, оптимизация). Желательно Postgres и ClickHouse.
- 1С как источник (ODBC/COM/API, структура типовых конфигураций).
- Matplotlib и др. визуализация (seaborn, plotly).
- Принципы BI-отчётности (метрики, дашборды, ad-hoc).

## Преимущество
Фарма; данные по движению лекарств (МДЛП, предметно-количественный учёт); аналитика по продажам,
остаткам, дистрибуции, аптекам, производству, маркетингу.
