# Вакансия: Старший продуктовый аналитик — hh.ru, discovery-команда (поиск и рекомендации)

hh.ru ищет senior продуктового аналитика в discovery-команду: улучшение поисковой
системы и алгоритмов рекомендаций. Data-driven компания, Data-informed решения.
Активно интегрируют AI-инструменты (Cursor, Claude, LobeHub, API моделей).
Стек: Hadoop, FineBI, Python, Airflow, своя A/B-платформа.

## Чем предстоит заниматься
- Исследовать эффективность изменений в поиске через анализ A/B-тестов; развивать
  A/B-платформу и методологию (повышение чувствительности, ГКГ, TDI, контроль пересечения
  экспериментов, сетевой эффект), другие фреймворки (региональные тесты, Causal Impact).
- Строить метрики и дашборды для точек роста; развивать определение нерелевантных
  элементов в выдачах; проводить и автоматизировать LLM-разметку поисковых запросов.
- Улучшать сегментацию пользователей; самостоятельные исследования взаимодействия
  пользователей с поиском; с менеджерами определять потенциал, риски, способы устранения.
- С монетизацией улучшать методологию оценки влияния инициатив поиска на выручку.

## Что поможет попасть
- Продуктовая аналитика от 3 лет.
- Отличный SQL (основной инструмент, превосходные навыки запросов).
- Опыт A/B + сильная теория: матстатистика, теория вероятностей.
- Python и основные библиотеки.
- Опыт с Apache Airflow.
- Дашборды: FineBI (приветствуется), Tableau или любая BI.
- Опыт с рекомендательными системами и анализом поисковых данных.
- Базовое ML, готовность к погружению в смежные области.
