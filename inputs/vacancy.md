# Вакансия: Аналитик по исследованиям (ИТ, финтех) — FIX

FIX — международная IT-компания и бизнес-инкубатор (финансы, e-commerce, digital-маркетинг).
Продукт: решение на базе поисковой аналитики и мониторинга — обнаруживает использование
платёжных инструментов финансовых организаций в нелегальных целях. Ищут аналитика, который
готовит аналитические документы для клиентов с фокусом на исследованиях и выводах. Удалённо.

## Основные задачи
- Поиск, сбор, верификация информации из разных источников: открытые данные, БД, отраслевые
  отчёты, СМИ, внутренние документы, телеграм-каналы, отраслевые сообщества.
- Анализ информации: ключевые тенденции, взаимосвязи, проблемы, тренды, закономерности.
- Изучение специфики отрасли и нормативной базы.
- Подготовка регулярных и разовых материалов: справок, отчётов, презентаций с выводами и рекомендациями.
- Визуализация данных и выводов (графики, диаграммы, инфографика).
- Мониторинг среды (рыночной, конкурентной, регуляторной) по направлениям.

## Требования
- Аналитическая должность от 3 лет (аналитик по исследованиям / в маркетинге / смежная).
- Большие объёмы информации из разных источников, критическая оценка достоверности.
- Методы качественного и количественного исследования.
- Аналитическое мышление + структурированное письмо, ясно излагать сложное.
- Свободный Excel (сводные, сложные формулы) и PowerPoint для презентаций.
- Использование ML/ИИ-инструментов для обработки массивов данных.
- Опыт подготовки аналитических документов (отчёты, справки, обзоры).
- Прикладное высшее образование.
- Внимательность, системный подход, ответственность за качество данных и выводов.
