# Вакансия: RWB (Wildberries + Russ) — аналитик данных (big data, SQL/DWH/BI)

RWB (Объединённая компания Wildberries и Russ) — международная технологическая компания
(слияние IT-компании Wildberries и оператора наружной рекламы Russ). (Работодатель назван — RWB.)

## Вам предстоит
- Анализ больших объёмов данных для выявления паттернов и тенденций под бизнес-решения.
- Создавать и оптимизировать SQL-запросы для извлечения, преобразования и загрузки данных (ETL).
- Исследовательская деятельность — анализ чувствительности и зависимостей данных.
- Выявлять возможности для автоматизации сбора и анализа данных.
- Создавать витрины данных в хранилищах (DWH) и BI-отчётность.

## Вы нам подходите, если
- Высшее образование в области математики, статистики или смежных областях.
- Глубокие знания SQL и опыт с реляционными базами данных.
- Опыт анализа больших объёмов данных и выводов на основе паттернов.
- Python для выгрузки и обработки больших данных.
- BI-инструменты (Tableau, Power BI, Metabase, Apache Superset).
- Уверенные знания Excel.
- Понимание принципов работы DWH.
