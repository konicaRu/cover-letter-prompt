# Вакансия: Senior Data Analyst — АО «Агропромцифра» (ГИС Минсельхоза)

АО «Агропромцифра» создано в 2023 году по директиве Правительства РФ для обеспечения информационной безопасности Минсельхоза РФ и подведомственных организаций. Единый поставщик Минсельхоза России по развитию ряда государственных информационных систем, осуществляет эксплуатацию ГИС. Ищут Senior Data Analyst. Проект предполагает сложные вычислительные алгоритмы (агрегации, прогнозные модели, скоринг, многомерные срезы) с требованиями к точности и производительности.

## Обязанности
- Самостоятельно проектировать витрины данных под аналитические задачи (MART-слой).
- Разрабатывать и оптимизировать сложные SQL-запросы (оконные функции, CTE, денормализация).
- Строить пайплайны агрегации из сырых логов и транзакционных данных.
- Настраивать проверки качества данных (Data Quality) на каждом этапе.
- Участвовать в обратной инженерии: анализировать существующие (иногда недокументированные) источники.
- Разрабатывать прототипы аналитических отчётов в Python (pandas, polars) или BI с последующей автоматизацией.

## Требования
- Опыт аналитических решений от 3 лет (Middle) / от 5 лет (Senior).
- SQL «свободно»: сложные join, оконные функции, оптимизация планов.
- Python базово: ETL-преобразование, aggregation, слияние датасетов (pandas/polars).
- Архитектура аналитической БД (ClickHouse, PostgreSQL, Greenplum — любой).
- Декомпозиция «хочу отчёт» → таблицы и расчёты.
- Профилирование производительности запросов.
- Бонусом: DWH (Kimball, Data Vault), Airflow или dbt.
