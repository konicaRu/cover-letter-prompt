## Ключевые «нельзя» по фактам резюме (C-008/C-009)
- НЕ упоминать ГНБ Москва / стройку / бурение / главного инженера и маркетинг/GA/Метрику оттуда.
- НЕ выдавать учебные проекты Я.Практикума (Афиша −11% и т.п.) за боевые результаты.
- НЕ заявлять то, чего нет в резюме: MS SQL/MySQL/Snowflake (есть ClickHouse, PostgreSQL), SSRS, SuperSet, OLAP-кубы, Airflow/SSIS/Spark/S3, Figma, 1С, OpenCV/CV, diff-in-diff/matching/IV, plotly, реферальные механики. Где требуется — честно признать отсутствие + мост через близкий реальный опыт (напр. SuperSet → Redash/Grafana).
