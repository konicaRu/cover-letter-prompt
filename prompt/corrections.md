# Правки (corrections)

Append-only. Не редактируй записи на месте — добавляй новые. Изменение состояния
правки делается только сменой её статуса.

## Статусы
- `active` — применяется при сборке промпта.
- `superseded-by C-NNN` — отменена более новой правкой, не применяется.
- `merged-into-core` — формулировка перенесена в core.md, здесь как история.

## Шаблон записи

```
### C-NNN
- Дата: YYYY-MM-DD
- Вакансия: <название/ссылка — где замечено>
- Триггер: <что в письме не устроило>
- Правило: <что делать или не делать — одна атомарная директива>
- Скоуп: always | <условие>
- Статус: active
```
